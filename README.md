# LifeLink
our comprehensive platform that connects donors, recipients, and healthcare providers to save lives through efficient organ and blood donation management.

The original project is available at https://www.figma.com/design/C18xszLeJ51uJqrrFDo8Bb/Organ---Blood-Donation-Platform.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
